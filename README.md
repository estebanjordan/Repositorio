# Portafolio - Julio Esteban Jordan Losada

Portafolio profesional desarrollado en React.

## Instalación

1. Crea una carpeta llamada `portfolio`
2. Copia todos los archivos en la estructura indicada
3. Abre la terminal en la carpeta del proyecto
4. Ejecuta:
```bash
npm install
```

## Ejecutar en desarrollo
```bash
npm start
```

La aplicación se abrirá en [http://localhost:3000](http://localhost:3000)

## Construir para producción
```bash
npm run build
```

Los archivos optimizados estarán en la carpeta `build/`

## Desplegar

Puedes desplegar en:
- **Vercel**: Conecta tu repositorio de GitHub
- **Netlify**: Arrastra la carpeta `build`
- **GitHub Pages**: Usa `gh-pages` para publicar

## Personalización

- **Datos personales**: Edita los componentes en `src/components/`
- **Estilos**: Modifica `src/styles/App.css`
- **Colores**: Cambia las variables CSS en `:root`
